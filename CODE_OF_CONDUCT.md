# Code of Conduct

## Our standard

This project is for local host security, skill governance, and defensive review. Participate in a way that is respectful, technically honest, and safe for operators and maintainers.

## Expected behavior

- share reproducible evidence
- keep reports privacy-safe
- disclose security concerns responsibly
- keep criticism concrete and focused on the work

## Unacceptable behavior

- posting secrets or private host data
- using the project to justify malware, persistence abuse, or credential theft
- publishing harassment, intimidation, or deceptive technical claims

## Enforcement

Project maintainers may remove unsafe content, close abusive reports, and restrict participation where needed to protect users and the repo.
