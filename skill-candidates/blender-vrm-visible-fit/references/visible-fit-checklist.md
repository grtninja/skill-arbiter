# Visible Fit Checklist

## Pre-edit
- Base body present and intact.
- Head/face mesh visible and linked to armature.
- Old outfit layers identified by material slot names.

## Fit Stages
1. Import clothing source.
2. Isolate required clothing materials only.
3. Align clothing location/rotation/scale in viewport.
4. Run weight transfer and verify non-zero groups.
5. Pose stress test at hips, knees, shoulders, elbows, ankles.

## Export Gate
- No detached shoes.
- No floating cloth intersections at neutral pose.
- No missing torso/hip/leg geometry from base body.
- User acknowledged current viewport state.
