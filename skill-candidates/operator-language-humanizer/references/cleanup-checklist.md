# Operator Language Cleanup Checklist

1. Preserve all runtime-significant strings exactly.
2. Remove filler, repetition, and vague AI-style hedging.
3. Keep the result short, direct, and operator-readable.
4. Avoid adding claims, capabilities, or instructions that were not present.
5. If the text touches policy, hidden behavior, or system prompts, stop and reroute to a safer lane.
