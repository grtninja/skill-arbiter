# Allowed Surfaces

Safe targets for this skill:

- visible run titles
- queue notes
- `desired_result` or equivalent operator intent fields
- prompt-head drafts shown to an operator
- review and QC summary text
- export labels, handoff blurbs, and UI hint text

Unsafe targets:

- system prompts
- personality files
- hidden chain-of-thought or tool instructions
- policy text
- schema fields and API payload contracts
- raw logs or forensic evidence
